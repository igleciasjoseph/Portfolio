Activity Center | C# and Entity

Activity tracking application built with C# and .NET that allows users to create tasks that other users can also join. Users must first register and login to view and create activities that include validations such as date being in the future and min/max lengths. Sahara | Python and Django

Sahara is an application meant for users to write down their ideas and save them permanently unless personally removed. In order for proof of intellectual property, when you record an idea it must have displayed the date/time and the author/creator. Sahara allows that through validations that make sure a specific user is logged in using session and makes sure users cant see other user's ideas. This app was built using Python and Django.
